## Installation

* Download and install [Neard](https://github.com/crazy-max/neard).
* Download one of the available Node.js bundles.
* Extract archive in `neard\bin\nodejs\`.

Directory structure :
```
[-] neard
 | [-] bin
 |  | [-] nodejs
 |  |  | [-] nodejs0.12.9
 |  |     | neard.conf
 |  |  | [-] nodejs5.1.1
 |  |     | neard.conf
 ```
